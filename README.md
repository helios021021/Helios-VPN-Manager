Nasıl Kullanılır?
ZIP dosyasını indirin ve bir klasöre çıkarın.

HeliosVPN.cmd dosyasını yönetici olarak çalıştırın.

Sunucu seçin ve "VPN AC" butonuna tıklayın.
(Zip şeklinde indirin, 'Engine'  klasörü ile ana dosya HeliosVPN.cmd aynı ortak klasörde olmak zorundadır aksi takdirde VPN çalışmayacaktır)



**How to Use**

1. Download the ZIP file and extract it to a folder.
2. Run **HeliosVPN.cmd** as Administrator.
3. Select a server and click the **"Connect VPN"** button.
(Download it as a ZIP file. The `Engine` folder and the main file, `HeliosVPN.cmd`, must be in the same directory; otherwise, the VPN will not work.)
